Development Notes

Thought Process

For the first level, I focused on creating a basic structure with a header, footer, and main content. Semantic elements such as <section> and <article> were used to enhance readability and organization.

The second level introduced a golden layout with two navigation menus and a table. I ensured proper structure without CSS to meet the requirements.

In the third level, I added dummy text, alt text for accessibility, and labels for table headers to improve the usability of the webpage.

Challenges Faced

  • Maintaining proper alignment and spacing without CSS was challenging but manageable with semantic HTML.
  • Ensuring accessibility required additional consideration for alt text and descriptive content.

End of Notes
